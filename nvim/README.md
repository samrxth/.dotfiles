# :zap: Installation

Prerequisites: **Git**, **Neovim v0.5**

1. Clone the github repository using the Github CLI.

```sh
git clone https://github.com/samrath2007/nvim ~/.config/nvim
```

2. Change to the project directory.

```sh
cd ~/.config/nvim
```

3. Installation commands

```sh
nvim + "PackerInstall"
```
